To start, run ./START.sh
All scripts should have exec permission.

Example folders included:
1) exampleFolderNoRepository - can be used to test creating a repository, by specifying path "exampleFolderNoRepository" when asked

2) exampleFolderWithRepository - can be used to test opening an existing repository, by specifying name "exampleRepo" when asked. Also showcases tracking changes of files.
NB: Only users with UID 1000 and 1001 can access ""exampleRepo" repository by default.
